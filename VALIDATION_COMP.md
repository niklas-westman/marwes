# Component Validation Checklist

Use this file as the step-by-step validation tracker for each Marwes component.

## Rule for every component attempt

Before validating any component, always start by running:

```text
/figma-sync refs marwes <component-name>
```

Then:

1. Open every file listed by `/figma-sync refs marwes <component-name>`
2. Compare the component implementation against the latest live sync in `.figma/marwes/`
3. Validate core, React, and Vue coverage where applicable
4. Update code if needed
5. Tick the checkbox when the component is validated
