## Summary

The Skeleton family is Marwes' structural loading-placeholder atom. It ships text, circular, and rectangular shapes with decorative default accessibility behavior and explicit standalone status mode through `ariaLabel`.

The current synced Figma source is V2/MUI-derived and mostly provides baseline shape geometry, so the shipped API stays intentionally small: shape, dimensions, radius, animation, and accessibility mode.
