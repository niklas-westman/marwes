## Summary

The Stat Tile family is Marwes' compact metric/KPI tile. It ships one focused surface with label, value, optional trend, optional subtitle, and five Figma-synced tone treatments.

The nested trend part supports positive and negative states. Arrow icons are decorative; the trend wrapper receives an accessible text label generated by core.
