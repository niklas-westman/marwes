# create-marwes

Vite-style starter wrapper for Marwes UI.

```bash
pnpm create marwes@latest my-app --template react-ts
pnpm create marwes@latest my-app --template vue-ts
pnpm create marwes@latest my-app --template svelte-ts
```

The starter delegates base app scaffolding to Vite, then runs the Marwes CLI init
flow for the selected adapter.

Use `--no-install` to skip dependency installation after the app is created.
