# Tooltip Components

Marwes provides the Tooltip family at two layers: the standalone **Tooltip** atom
for a visible bubble, and **TooltipGroup** for the Figma-matched help-icon
trigger that reveals the bubble on hover and focus.

## Base: Tooltip (Atom)

Use `Tooltip` when surrounding layout already owns placement or visibility.
This maps directly to the standalone tooltip bubble in the synced Figma files.

## Recommended: TooltipGroup (Molecule)

Use `TooltipGroup` for the default contextual-help pattern from Figma.
It renders a small icon trigger and shows the tooltip bubble on hover and focus.

## Component Reference

- `Tooltip` — visible tooltip bubble using the inverse surface token
- `TooltipGroup` — help-icon trigger with hover and focus reveal behavior
- `IconName.HelpCircle` — default trigger icon
- `IconName.Info` — swapped trigger icon from the Figma tooltip group examples
