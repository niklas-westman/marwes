# Skeleton Component

Marwes provides `Skeleton` as a low-level placeholder atom for loading text, circular media, and rectangular content blocks.

The synced Skeleton page provides three baseline shapes: text `120×12`, circular `40×40`, and rectangular `120×120`.

Skeletons are decorative by default. Provide `ariaLabel` when a standalone skeleton group must announce loading status.
