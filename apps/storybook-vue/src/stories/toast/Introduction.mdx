# Toast Components

Toasts communicate transient status updates without interrupting workflow.

## Component Architecture

```
Toast (Atom)
```

## Recommended Usage

Use `Toast` for confirmation, warning, and action follow-up feedback.

```vue
<script setup lang="ts">
import { Toast } from "@marwes-ui/vue"

const onDismiss = () => {}
</script>

<template>
  <Toast variant="rich" ariaLive="polite" :onDismiss="onDismiss">
    Your changes have been saved.
  </Toast>
</template>
```
