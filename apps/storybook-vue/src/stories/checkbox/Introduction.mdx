# Checkbox Components

Checkbox stories are organized with a simple two-layer structure to keep discovery fast.

## Checkbox Architecture

```
Checkbox (Atom)
     ↓        ↓
CheckboxField (Molecule)
CheckboxGroupField (Molecule)
```

## Atom: Checkbox

Use `Checkbox` for low-level control when you manage label/layout yourself.

```vue
<script setup lang="ts">
import { Checkbox } from "@marwes-ui/vue"
</script>

<template>
  <label>
    <Checkbox ariaLabel="Accept terms" />
    Accept terms
  </label>
</template>
```

## Molecule: CheckboxField

Use `CheckboxField` when you want built-in field semantics and accessibility wiring.

```vue
<script setup lang="ts">
import { CheckboxField } from "@marwes-ui/vue"
</script>

<template>
  <CheckboxField
    label="Subscribe to updates"
    description="We'll only send important product updates."
    :checkbox="{ size: 'md' }"
  />
</template>
```

## Molecule: CheckboxGroupField

Use `CheckboxGroupField` when you need a labeled multi-select checkbox list with shared helper/error text.

```vue
<script setup lang="ts">
import { CheckboxGroupField } from "@marwes-ui/vue"
</script>

<template>
  <CheckboxGroupField
    label="Notification preferences"
    description="Choose every channel you want us to use."
    :options="[
      { value: 'email', label: 'Email updates' },
      { value: 'sms', label: 'SMS alerts' },
      { value: 'push', label: 'Push notifications' },
    ]"
  />
</template>
```
