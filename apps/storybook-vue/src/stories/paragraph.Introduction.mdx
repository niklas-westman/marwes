# Paragraph

Paragraph is the baseline text primitive for body copy and supporting content.

## Component Architecture

```
Paragraph (Atom)
```

## Recommended Usage

Default to `md`; use `sm` for secondary text and `lg` for lead text.

```vue
<script setup lang="ts">
import { Paragraph } from "@marwes-ui/vue"
</script>

<template>
  <Paragraph size="md">Body text content.</Paragraph>
</template>
```

## Story Coverage

- All size variants (`sm`, `md`, `lg`)
- Long-form copy and article composition
- Custom class usage and spacing patterns
