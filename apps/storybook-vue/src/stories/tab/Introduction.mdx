# Tab Components

Tabs provide segmented navigation between peer content sections.

## Component Architecture

```
Tab (Atom)
```

## Recommended Usage

Use `Tab` inside a `role="tablist"` with coordinated selected state and panel wiring.

```vue
<script setup lang="ts">
import { ref } from "vue"
import { Tab } from "@marwes-ui/vue"

const active = ref(0)
</script>

<template>
  <div role="tablist" aria-label="Profile sections" style="display: flex;">
    <Tab :selected="active === 0" ariaControls="panel-overview" @click="active = 0">Overview</Tab>
    <Tab :selected="active === 1" ariaControls="panel-activity" @click="active = 1">Activity</Tab>
  </div>
</template>
```

## Story Coverage

- `Default`, `Selected`, and `Disabled` for state validation
- `TabList` for interactive row behavior
- `AllStates` for quick visual regression checks
