# Pagination

Marwes provides `Pagination` for navigating finite page sets while keeping page
windowing, ellipses, and active-page semantics consistent across frameworks.

## Component Reference

Use `modelValue` with `update:modelValue` in Vue, or provide `defaultPage` for
uncontrolled usage.
