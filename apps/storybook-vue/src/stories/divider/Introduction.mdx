# Divider Component

Marwes provides Divider as a single atom for separating related content without
introducing extra structure. Use it to create visual breaks between stacked
sections or as a vertical separator in inline layouts.

## Component Architecture

```
Divider (Atom)
```

## Recommended Usage

Use the raw `Divider` atom directly. Choose horizontal dividers for stacked
content and vertical dividers inside flex or inline layouts.

```vue
<script setup lang="ts">
import { Divider } from "@marwes-ui/vue"
</script>

<template>
  <Divider size="sm" />
  <Divider orientation="vertical" size="xs" />
</template>
```

## Usage Notes

- `size` controls both divider thickness and the built-in surrounding spacing
- `orientation="horizontal"` is the default for content sections
- `orientation="vertical"` works best in containers that define their own height

## Component Reference

**`Divider`** - Low-level separator atom for horizontal or vertical content
boundaries
