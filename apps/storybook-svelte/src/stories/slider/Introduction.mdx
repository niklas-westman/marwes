# Slider Components

Marwes provides a slider system with three layers:
**purpose sliders** for common control patterns, **SliderField** for labeled form usage,
and the raw **Slider** atom for custom layouts.

## Component Architecture

```
Slider (Atom)
     ↓
SliderField (Molecule)
     ↓
Purpose Wrappers
  ├─ VolumeSlider
  ├─ BrightnessSlider
  └─ RadiusSlider
```

## Recommended: Purpose Components

Use purpose-built sliders when the control has a clear product meaning:

```svelte
<script>
  import { VolumeSlider, BrightnessSlider, RadiusSlider } from "@marwes-ui/svelte"
</script>

<VolumeSlider value={45} />
<BrightnessSlider value={75} />
<RadiusSlider value={16} />
```

## Base: SliderField (Molecule)

Use `SliderField` for forms and settings panels:

```svelte
<script>
  import { SliderField } from "@marwes-ui/svelte"
  let radius = $state(24)
</script>

<SliderField
  label="Radius"
  helperText="Controls the corner rounding used by the UI."
  slider={{ min: 0, max: 48, step: 2, showTooltip: true }}
  bind:value={radius}
/>
```

## Low-level: Slider (Atom)

Use `Slider` directly when you need a custom composition:

```svelte
<script>
  import { Slider } from "@marwes-ui/svelte"
  let volume = $state(50)
</script>

<Slider min={0} max={100} bind:value={volume} showTooltip />
```

## Supported Features

- `min`, `max`, and `step` define the numeric range
- `bind:value` for two-way binding
- `onvaluechange` callback for value updates
- `showTooltip` displays the current numeric value above the thumb
- `disabled` preserves native disabled semantics
- Custom visual track with fill, thumb, and touch-area elements

## Accessibility Notes

- Renders a native `<input type="range">` for keyboard support
- Provide `ariaLabel` when there is no visible label
- Use SliderField when the slider needs visible label, helper text, or error

## Component Reference

**`VolumeSlider`** — Audio and media output levels (`data-purpose="volume"`)

**`BrightnessSlider`** — Visual intensity adjustments (`data-purpose="brightness"`)

**`RadiusSlider`** — Corner rounding controls (`data-purpose="radius"`)

**`SliderField`** — Field wrapper with label, helper/error, and min/max values

**`Slider`** — Low-level range input atom with visual track
