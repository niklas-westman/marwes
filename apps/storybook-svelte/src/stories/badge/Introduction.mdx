# Badge Components

Marwes provides badges at three layers: **purpose components** for common
status contexts, **BadgeGroup** for grouped layouts, and the raw **Badge**
atom for custom composition.

## Component Architecture

```
Badge (Atom)
     ↓
BadgeGroup (Molecule)
     ↓
Purpose Wrappers
  ├─ StatusBadge
  ├─ PriorityBadge
  └─ NotificationBadge
```

## Recommended: Purpose Components

Use purpose-built badges when the badge has a clear semantic meaning:

```svelte
<script>
  import { StatusBadge, PriorityBadge, NotificationBadge } from "@marwes-ui/svelte"
</script>

<StatusBadge variant="success">Active</StatusBadge>
<PriorityBadge variant="warning">High</PriorityBadge>
<NotificationBadge variant="info" ariaLabel="5 unread messages">5</NotificationBadge>
```

## Base: BadgeGroup (Molecule)

For grouped badge layouts:

```svelte
<script>
  import { Badge, BadgeGroup } from "@marwes-ui/svelte"
</script>

<BadgeGroup label="Tags">
  <Badge variant="info">Frontend</Badge>
  <Badge variant="success">Approved</Badge>
  <Badge variant="warning">Review</Badge>
</BadgeGroup>
```

## Atom: Badge

Use the raw `Badge` atom for standalone status indicators:

```svelte
<script>
  import { Badge } from "@marwes-ui/svelte"
</script>

<Badge variant="neutral">Neutral</Badge>
<Badge variant="info">Info</Badge>
<Badge variant="success">Success</Badge>
<Badge variant="warning">Warning</Badge>
<Badge variant="error">Error</Badge>
```

## Usage Notes

- `variant` controls semantic color: `neutral`, `info`, `success`, `warning`, `error`
- Badge content is passed as children
- `ariaLabel` provides an accessible name for icon-only or abbreviated badges

## Component Reference

### Purpose Components

**`StatusBadge`** — Operational status indicators (`data-purpose="status"`)

**`PriorityBadge`** — Task or issue priority labels (`data-purpose="priority"`)

**`NotificationBadge`** — Unread counts and alerts (`data-purpose="notification"`)

### Molecule

**`BadgeGroup`** — Grouped badge layout with accessible fieldset

### Atom

**`Badge`** — Inline status indicator with semantic color variants
