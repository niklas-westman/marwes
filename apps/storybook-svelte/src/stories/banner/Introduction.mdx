# Banner

A persistent, page-level status message component for inline announcements,
system updates, or required actions.

## Usage

```svelte
<script lang="ts">
  import { Banner, Button } from "@marwes-ui/svelte"
</script>

<!-- Basic info banner -->
<Banner variant="info">
  Your account has been updated successfully.
</Banner>

<!-- With CTA action -->
<Banner variant="warning">
  Your subscription expires in 3 days.
  {#snippet action()}
    <Button size="sm">Review</Button>
  {/snippet}
</Banner>

<!-- Dismissible with callback -->
<Banner variant="success" ondismiss={() => visible = false}>
  Changes saved.
</Banner>

<!-- Without icon -->
<Banner variant="neutral" showIcon={false}>
  Maintenance scheduled for tonight.
</Banner>
```

## Variants

| Variant | Use for | a11y role |
| --- | --- | --- |
| `neutral` | General announcements | `status` (polite) |
| `info` | Informational messages | `status` (polite) |
| `success` | Positive confirmations | `status` (polite) |
| `warning` | Caution, upcoming deadlines | `alert` (assertive) |
| `error` | Errors, critical issues | `alert` (assertive) |

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `variant` | `BannerVariant` | `"neutral"` | Visual intent |
| `showIcon` | `boolean` | `true` | Show leading icon |
| `dismissible` | `boolean` | `true` | Show close button |
| `ondismiss` | `() => void` | — | Close button callback |
| `ariaLabel` | `string` | — | Accessible label for the banner region |

## Snippets

| Snippet | Description |
| --- | --- |
| `children` | Banner message content (default) |
| `action` | CTA action content |
| `icon` | Custom icon (defaults to info circle) |

## When to use

- **Banner**: Persistent page-level messages that stay until dismissed
- **Toast**: Transient feedback that auto-dismisses
- **Dialog**: Blocking confirmations that require user action
