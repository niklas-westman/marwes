# Divider Component

Marwes provides `Divider` as a single atom for separating related content without
introducing extra structure. Use it to create visual breaks between stacked
sections or as a vertical separator in inline layouts.

## Component Architecture

```
Divider (Atom)
```

## Recommended Usage

Use the raw `Divider` atom directly:

```svelte
<script>
  import { Divider } from "@marwes-ui/svelte"
</script>

<Divider size="sm" />
<Divider orientation="vertical" size="xs" />
```

## Usage Notes

- `size` controls both divider thickness and surrounding spacing
- `orientation="horizontal"` is the default for content sections
- `orientation="vertical"` works best in containers that define their own height

## Accessibility notes

**Divider is a real semantic separator.** It renders an `<hr>` with separator semantics.
Use it when the UI benefits from an actual content boundary, not just a visual line.

**Do not use Divider as a spacing helper.** If the layout only needs whitespace, use
the `Spacing` component instead.

## Component Reference

**`Divider`** — Low-level separator atom for horizontal or vertical content boundaries
