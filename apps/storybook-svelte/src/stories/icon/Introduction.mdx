# Icon Component

Marwes provides `Icon` as an atom-level primitive for rendering system icons
at consistent sizes.

## Component Architecture

```
Icon (Atom)
```

## Recommended Usage

Use the raw `Icon` atom with a named icon:

```svelte
<script>
  import { Icon } from "@marwes-ui/svelte"
</script>

<Icon name="heart" size="md" />
<Icon name="search" size="sm" />
<Icon name="settings" size="lg" />
```

## Usage Notes

- `name` selects the icon from the Marwes icon set
- `size` supports `sm`, `md`, `lg`
- Icons are decorative by default; add `ariaLabel` when the icon carries standalone meaning

## Accessibility notes

**Decorative icons are hidden from assistive technology.** When an icon accompanies
visible text, leave it unlabeled. Only add `ariaLabel` when the icon is the sole
communicator of meaning (e.g., an icon-only button).

## Component Reference

**`Icon`** — Inline icon atom from the Marwes icon set
