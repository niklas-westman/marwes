# Checkbox Component

Marwes provides `Checkbox` as an atom-level toggle input for binary or
indeterminate selections.

## Component Architecture

```
Checkbox (Atom)
```

## Recommended Usage

Use the raw `Checkbox` atom for individual selection controls:

```svelte
<script>
  import { Checkbox } from "@marwes-ui/svelte"
</script>

<Checkbox ariaLabel="Accept terms" />
<Checkbox checked ariaLabel="Already accepted" />
<Checkbox indeterminate ariaLabel="Select all" />
```

## Usage Notes

- `checked` controls the checked state
- `indeterminate` shows a mixed-state indicator
- `disabled` prevents interaction
- `ariaLabel` provides an accessible name when no visible label is present

## Accessibility notes

**Always provide a label.** Use `ariaLabel` when the checkbox is not wrapped in a
visible label element. Without a name, assistive technology cannot communicate
the control's purpose.
