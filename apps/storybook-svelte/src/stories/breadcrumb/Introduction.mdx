# Breadcrumb

Marwes provides `Breadcrumb` for showing a compact page hierarchy with a home affordance, link items, chevron separators, and a current page marker.

## Usage

```svelte
<script>
  import { Breadcrumb } from "@marwes-ui/svelte"
</script>
```

## Accessibility

- Uses a labelled `nav` landmark
- Uses ordered list semantics
- Marks the current item with `aria-current="page"`
