# StatTile Component

Marwes provides `StatTile` as an atom-level data display for key metrics
with optional trend indicators and semantic tone.

## Component Architecture

```
StatTile (Atom)
```

## Recommended Usage

```svelte
<script>
  import { StatTile } from "@marwes-ui/svelte"
</script>

<StatTile label="Revenue" value="$12,345" />

<StatTile
  label="Users"
  value="1,234"
  trendValue="+12%"
  trendDirection="positive"
/>

<StatTile
  label="Churn"
  value="3.2%"
  trendValue="-0.5%"
  trendDirection="negative"
  tone="danger"
/>

<StatTile label="Revenue" value="$12,345" subtitle="Last 30 days" />
```

## Props

- `label` — Describes the metric (required)
- `value` — Primary figure display (required)
- `trendValue` — Change indicator text (e.g. "+12%", "-0.5%")
- `trendDirection` — `"positive"` or `"negative"` — controls icon and color
- `tone` — Semantic color: `"danger"` for negative metrics
- `subtitle` — Additional context below the value

## Usage Notes

- Renders as `<article>` for semantic grouping
- Trend icon appears automatically based on `trendDirection`
- Use `tone="danger"` for metrics where negative values are bad
- `subtitle` provides secondary context (time period, comparison)

## Accessibility Notes

- The trend region has `aria-label` describing the change direction and value
- Trend icons are decorative (`aria-hidden="true"`)
