# Spinner Component

Marwes provides `Spinner` as an atom-level loading indicator with multiple
visual variants synced from Figma.

## Component Architecture

```
Spinner (Atom)
```

## Recommended Usage

Use the raw `Spinner` atom for inline or standalone loading feedback:

```svelte
<script>
  import { Spinner } from "@marwes-ui/svelte"
</script>

<Spinner variant="classic" size="md" />
<Spinner variant="dots-round" size="lg" />
<Spinner variant="ring" size="sm" />
```

## Usage Notes

- `variant` maps to seven synced Spinner styles: `classic`, `ring`, `dual`, `dots-round`, `dots-square`, `lines`, `cross`
- `size` supports `sm`, `md`, `lg`
- Keep the spinner decorative when nearby text already communicates loading
- Add `ariaLabel` when the spinner must announce loading on its own

## Accessibility notes

**Decorative by default.** When visible text nearby already says what is loading,
the spinner should remain hidden from assistive technology.

**Reduced-motion.** The spinner animation slows under `prefers-reduced-motion: reduce`
rather than stopping entirely, keeping visible loading feedback present.

## Component Reference
