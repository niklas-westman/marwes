# Pagination

Marwes provides `Pagination` for navigating finite page sets while keeping page
windowing, ellipses, and active-page semantics consistent across frameworks.

## Component Reference

Use `bind:page` in Svelte for two-way page state, or pass `defaultPage` for an
uncontrolled starting page.

Use `showFirstLast` for large ranges, `getItemAriaLabel` for localized control
labels, and the default adaptive behavior for narrow containers. Use
`controlDisplay="auto" | "label" | "icon"` to let previous/next labels collapse
responsively or to force a specific control treatment.
