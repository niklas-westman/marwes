# Heading

Headings provide semantic structure and visual hierarchy for page content.

## Component Architecture

```
H1/H2/H3 (Atoms)
```

## Recommended Usage

Choose heading level by document structure first, then use size overrides only when needed.

```tsx
import { H1, H2, H3 } from "@marwes-ui/react"

<H1>Page Title</H1>
<H2>Section</H2>
<H3>Subsection</H3>
```
