# Breadcrumb

Marwes provides `Breadcrumb` for showing a compact page hierarchy with a home affordance, link items, chevron separators, and a current page marker.

## When to use

- Detail pages nested inside a larger information architecture
- Dashboards or documentation where users need quick parent navigation
- Compact secondary navigation above page content

## Usage

```tsx
import { Breadcrumb } from "@marwes-ui/react"

<Breadcrumb
  homeHref="/"
  items={[
    { label: "Components", href: "/components" },
    { label: "Breadcrumb" },
  ]}
/>
```

## Accessibility

- Uses a labelled `nav` landmark
- Uses ordered list semantics for hierarchy
- Marks the current item with `aria-current="page"`
- Keeps the home icon labelled for screen readers
