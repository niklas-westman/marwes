# Toast Components

Toasts communicate transient status updates without interrupting workflow.

## Component Architecture

```
Toast (Atom)
```

## Recommended Usage

Use `Toast` for confirmation, warning, and action follow-up feedback.

```tsx
import { Toast } from "@marwes-ui/react"

<Toast variant="rich" ariaLive="polite" onDismiss={() => {}}>
  Your changes have been saved.
</Toast>
```

## Story Coverage

- `Default`, `WithDismiss`, and `WithAction` for core behaviors
- `AllVariants` and `DarkVariants` for style parity
- `Urgent` for assertive announcements
