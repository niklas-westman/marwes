# Badge Components

Badges communicate compact status and metadata in a low-footprint UI element.

## Component Architecture

```
Badge (Atom)
```

## Recommended Usage

Use `Badge` for short, meaningful labels such as state, category, or severity.

```tsx
import { Badge } from "@marwes-ui/react"

<Badge variant="success">Active</Badge>
```
