# Divider

Dividers separate related content blocks and improve scanability.

## Component Architecture

```
Divider (Atom)
```

## Recommended Usage

Use horizontal dividers between stacked sections and vertical dividers in inline layouts.

```tsx
import { Divider } from "@marwes-ui/react"

<Divider size="sm" />
```
