# Icon Gallery

The icon gallery is a discovery surface for all available icon tokens.

## Component Architecture

```
Icon (Atom)
```

## Recommended Usage

Search by name, then validate size and stroke width in context.

```tsx
import { Icon } from "@marwes-ui/react"

<Icon name="search" size="sm" strokeWidth="md" />
```

## Story Coverage

- Full icon inventory with live filtering
- Size and stroke width controls
- Responsive grid density controls
